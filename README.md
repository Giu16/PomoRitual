# PomoRitual

> Um Pomodoro Timer focado em produtividade consciente, acessibilidade e experiência fluida.

O PomoRitual é uma aplicação front-end construída em JavaScript puro (Vanilla JS) que implementa ciclos de foco e pausa utilizando manipulação dinâmica de DOM, controle de estado e integração com Web APIs.

O projeto foi desenvolvido como parte da consolidação dos meus fundamentos em desenvolvimento web moderno, priorizando:

-  Responsividade
-  Acessibilidade (A11Y)
-  Performance
-  Organização de código
-  Experiência do usuário

## ✨ Funcionalidades

✔️ Timer Pomodoro (Foco + Pausa)  
✔️ Controle de início, pausa e reset  
✔️ Alertas sonoros sincronizados  
✔️ Notificações nativas via Web Notifications API  
✔️ Interface responsiva  
✔️ Navegação por teclado (Acessibilidade)  
✔️ Feedback visual com SVG animado  

> ⚠️ Usuários iOS: utilizar Safari para funcionamento completo das notificações.

---

## 🧠 Sobre o Método Pomodoro

O método Pomodoro é uma técnica de gestão de tempo que alterna ciclos intensos de foco com pausas estratégicas.

Benefícios:
- 📈 Aumento da produtividade
- 🧘 Redução da fadiga mental
- 🎯 Melhora na concentração
- 🔁 Consistência no ritmo de trabalho
